# helpful info for using the ``nano`` text-editor in Linux

## text manipulation

* copy -> ``tbd``
* paste -> ``Ctrl-v | Ctrl-Shift-v``
* cut / uncut -> ``Ctrl-k / Ctrl-u``
* undo -> ``Alt-u``

## configuration -> added to ``.nanorc`` file

* enable mouse support -> ``set mouse``
* enable line numbers -> ``set linenumbers``
